
- Contact Section
Dibuat form kontak responsif dengan validasi.
Ada notifikasi saat form berhasil dikirim.
Menggunakan struktur HTML5 yang aksesibel.

- Testimonial Section
Dibuat carousel testimonial dengan tombol navigasi.
Menggunakan array data untuk menyimpan beberapa testimonial.
Konten berubah dinamis dengan transisi halus.

- Portfolio Section
Grid responsif untuk item portofolio.
Hover effect, gradient overlay, dan desain kartu.

- Penjelasan Carousel
Menggunakan array objek testimonial (nama, role, teks, avatar, gambar).
Tombol prev/next mengubah indeks aktif.
Fungsi updateTestimonial() memperbarui tampilan tanpa reload.
Navigasi melingkar (infinite loop) dengan modulo.

- Prinsip Desain
Layout pakai CSS Grid & Flexbox.
Tema konsisten dengan CSS variables.
Hover effect, gradient, shadow.
Typography: Junicode (body), Bricolage Grotesque (judul).
